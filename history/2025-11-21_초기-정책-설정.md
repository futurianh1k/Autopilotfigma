## 📝 핵심 정책 요약

### 1️⃣ 언어 정책
- ✅ 모든 응답 한국어 제공
- ✅ 코드 주석 한국어 작성
- ✅ 커밋 메시지 한국어 우선

### 2️⃣ 참고 자료 명시
```typescript
/**
 * @library 라이브러리명 (버전)
 * @reference 공식 문서 URL
 * @license 라이선스 타입
 */
```

### 3️⃣ 수정 내용 표시
- 파일 경로
- 변경 이유
- 수정 전/후 비교
- 영향 범위

### 4️⃣ 테스트 코드
- 모든 신규 기능에 테스트 작성
- `*.test.tsx` 또는 `*.test.ts` 파일 생성
- Vitest + React Testing Library 사용

### 5️⃣ 브랜치 전략
```
feature/<기능명>      - 신기능
db/<변경내용>        - DB 수정
ui/<변경내용>        - UI 대규모 변경
refactor/<변경내용>  - 아키텍처 변경
```

### 6️⃣ 변경 이력
- `history/YYYY-MM-DD_<제목>.md` 형식
- 사용자 요구사항, AI 응답, 수정 내용 포함
